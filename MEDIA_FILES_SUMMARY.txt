FEATURE SPECIFICATIONS
═══════════════════════════════════════════════════════════════════════════════

Media Upload Constraints:
- Maximum files per appointment: 5
- Maximum file size: 25MB
- Allowed image types: JPG, PNG, WEBP
- Allowed video types: MP4, MOV
- Allowed audio types: MP3, M4A

UI Location: BookingConfirmationModal (before payment step)

Trigger Point: After appointment creation, before Razorpay payment

Optional: Yes (doesn't block booking flow)

Storage: Cloudinary CDN

Authorization:
- Patient: Own appointments only (pending/confirmed status)

- Admin: All media access
